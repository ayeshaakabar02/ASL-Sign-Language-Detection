This project focuses on building an Automatic American Sign Language (ASL) Detection system using Deep Learning. The system is designed to recognize hand signs from images and predict the corresponding ASL alphabet.
The main goal of this project is to bridge the communication gap between hearing-impaired individuals and others by using modern AI techniques. The complete project is developed and tested in Visual Studio Code, and the final deployment is done using Streamlit, which provides an interactive and user-friendly web interface.
